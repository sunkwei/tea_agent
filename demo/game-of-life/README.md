# 🌱 康威生命游戏 · Game of Life

**John Conway (1970)** — 四条简单规则构成的图灵完备宇宙。

## 规则

每个细胞的状态由周围 8 个邻居决定：

| 当前状态 | 条件 | 结果 |
|---------|------|------|
| 💀 死 | 周围恰有 **3** 个活细胞 | ➡️ **诞生** |
| 🟢 活 | 周围 **2 或 3** 个活细胞 | ➡️ **存活** |
| 🟢 活 | 周围 **< 2** 个活细胞 | ➡️ **孤独死亡** |
| 🟢 活 | 周围 **> 3** 个活细胞 | ➡️ **拥挤死亡** |

## 预设图案

| 图案 | 行为 | 说明 |
|------|------|------|
| 🧱 **方块 (Block)** | 静止 | 稳定结构，永不变化 |
| ⏳ **闪烁器 (Blinker)** | 周期振荡 | 3 格横竖交替，周期 2 |
| 🦅 **滑翔机 (Glider)** | 平移运动 | 每 4 步向右下移动一格 |
| 🔫 **滑翔机枪 (Gosper Gun)** | 无限发射 | 持续产生滑翔机，无限增殖 |
| ⭐ **脉冲星 (Pulsar)** | 周期振荡 | 周期 3 的复杂对称振荡器 |
| 🎲 **随机** | 混沌 | 初始随机填充 25%，观察混沌演化 |

## 玩法

| 操作 | 说明 |
|------|------|
| ▶️ **运行** | 连续演化 |
| ⏭️ **单步** | 逐代前进 |
| 🖱️ **点击画布** | 创造（绿）或抹除生命 |
| 🔄 **清空** | 重置棋盘 |
| **预设图案** | 一键加载经典图案 |

## 数学背景

### 1. 图灵完备
生命游戏可以构造与门、或门、非门、时钟和存储器——理论上可以模拟任意计算机。已有实现：CPU + 操作系统 + 俄罗斯方块。

### 2. 不可预测性
给定初始图案，无法在不实际运行的前提下预测其 N 代后的状态。计算本身是最快的模拟方式。

### 3. 自复制
某些图案（如 Langton's Loop）可以自我复制——无限繁衍。
