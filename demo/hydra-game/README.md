# 🐍 九头蛇游戏 · Hydra Game

**Kirby-Paris Hydra Game (1982)** — 一个看起来无限增长、实际上必然消灭的数学奇观。

## 游戏规则

1. 九头蛇是一棵**根树**，每个叶子节点是一个**蛇头**
2. 每步砍掉一个蛇头（点击红色节点）
3. 若蛇头的父节点是**根** → 直接消失，不再生
4. 否则，从**祖父节点**处生长 **N** 份父子树副本（**N = 当前步数**）
5. 蛇头越多，长得越猛，但**树的深度只降不升**
6. 最终蛇头会被全部砍光——**虽然无法用皮亚诺算术证明，但必然在有限步内结束**

## 玩法

| 操作 | 说明 |
|------|------|
| 🖱️ **点击红色蛇头** | 砍掉该蛇头，观察再生 |
| ▶️ **自动演示** | 自动砍头，观察完整过程 |
| ⏱️ **速度滑块** | 调节自动演示速度 |
| 🔄 **重开** | 重新开始 |

## 统计指标

| 指标 | 含义 |
|------|------|
| **步数** | 已砍头次数 |
| **蛇头数** | 当前叶子节点数量 |
| **总节点** | 树中所有节点数 |
| **最大深度** | 树的层数（此值永不增加） |

## 数学背景

九头蛇游戏由 **Laurie Kirby** 和 **Jeff Paris** 在 1982 年提出，目的是给出 **古德斯坦定理（Goodstein's Theorem）** 的一个直观实例。

**核心结论**：游戏保证在有限步内结束——但这不能用皮亚诺算术（PA）证明，需要更强的集合论公理（超限归纳法到 ε₀）。

这是**哥德尔不完备定理**的一个具体实例：一个在标准自然数模型中为真、但在皮亚诺算术中不可证明的命题。
