# 🐜 朗顿的蚂蚁 · Langton's Ant

**Chris Langton (1986)** — 简单确定性规则下涌现出无限复杂的秩序。

## 规则

蚂蚁在无限网格上爬行，每一步：

| 当前格子颜色 | 动作 |
|------------|------|
| ⬜ **白色** | 右转 90°，涂黑，前进一格 |
| ⬛ **黑色** | 左转 90°，涂白，前进一格 |

## 现象

蚂蚁的行为分为三个阶段：

| 阶段 | 步数范围 | 现象 |
|------|---------|------|
| 🌪️ **混沌** | 0 ~ 200 步 | 杂乱无章的花纹，无规律 |
| 🌀 **过渡** | 200 ~ 10,000 步 | 对称结构出现又消失，混沌与秩序交织 |
| 🛣️ **高速公路** | ≈10,000 步+ | 蚂蚁沿直线笔直走下去，永不回头 |

## 玩法

| 操作 | 说明 |
|------|------|
| ▶️ **运行** | 连续运行，观察三个阶段的演变 |
| ⏭️ **单步** | 逐帧前进，仔细看每一步的变化 |
| ⏱️ **速度** | 调节运行速度 |
| 🔄 **重置** | 重新开始 |

## 数学背景

### 1. 涌现（Emergence）
两条只看"脚下"的局部规则，整体上却产生了秩序井然的"高速公路"。这种**局部 → 全局**的突现现象在自然界广泛存在（蚁群、神经系统、经济市场）。

### 2. 计算普适性
Langton 证明了蚂蚁的行为等价于一个通用图灵机——理论上它可以执行任何计算。一个最简单的循环规则，等价于万能计算机。

### 3. 确定性混沌
系统是完全确定的（没有随机数），但长期行为**不可预测**。你无法在不实际运行的前提下知道高速公路何时出现——这体现了**确定性系统的不可约简性**。

### 4. 高速公路的证明
虽然现象在 1986 年就被发现，但直到 **2020 年** 才被 A. Gajardo、E. Goles 等人严格证明：所有初始状态下，蚂蚁最终一定会进入周期性轨道（高速公路）。
