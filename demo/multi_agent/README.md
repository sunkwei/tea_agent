# 🥊 Multi-Agent 辩论赛

双 Agent 对抗辩论服务器——两个独立 AI 模型进行实时多轮辩论，前端通过 SSE 流式展示。

## 快速开始

```bash
# 安装依赖
pip install starlette uvicorn

# 启动辩论服务器
cd demo
python multi_agent/server.py --port 8083

# 浏览器打开
# http://127.0.0.1:8083
```

## 工作原理

```
┌──────────────┐     SSE 流      ┌──────────────┐
│  前端 UI     │◄───────────────│  Starlette   │
│  index.html  │                │  Server      │
└──────────────┘                └──────┬───────┘
                                       │
                          ┌────────────┼────────────┐
                          │            │            │
                     ┌────▼───┐  ┌────▼───┐        │
                     │ 甲方    │  │ 乙方    │        │
                     │ Agent  │  │ Agent  │        │
                     └────────┘  └────────┘        │
                          │            │            │
                     ┌────▼────────────▼────┐       │
                     │  辩论上下文（50轮）   │       │
                     │  每方可见对方全部发言  │       │
                     └──────────────────────┘       │
```

## 辩论流程

1. **甲方开篇立论**（第 1 轮）
2. **乙方反驳**（第 2 轮）
3. **双方轮换**（第 3-50 轮）
4. 每方都能看到对方之前的所有发言
5. 前端实时展示，支持取消辩论

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/` | 辩论 UI 页面 |
| GET | `/api/configs` | 列出可用的 AI 配置 |
| POST | `/api/debate/start` | 启动辩论（`{"topic":"..."}`） |
| POST | `/api/debate/cancel` | 取消辩论 |

## 技术栈

- **后端**: Starlette + SSE 流式推送
- **前端**: 原生 HTML + CSS + JavaScript
- **Agent**: tea_agent OnlineToolSession，双 Session 独立运行
- **最大轮数**: 50 轮
