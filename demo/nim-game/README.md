# 🎯 尼姆游戏 · Nim

**组合博弈论的基石** — 一个 XOR 决定胜负的数学游戏。

## 规则

- 若干堆石子，每堆数量任意
- 每轮选 **一堆**，取走 **任意正数颗**（可以全取走）
- **取走最后一颗石子的人获胜**

## 玩法

| 操作 | 说明 |
|------|------|
| 🖱️ **点击堆** | 选中要操作的石堆 |
| 🔢 **输入数量** | 输入取走的石子数 |
| 🤖 **AI 对战** | AI 使用最优 XOR 策略 |
| 🔄 **重开** | 重新随机布阵 |

## 设置

| 选项 | 说明 |
|------|------|
| **堆数** | 3~5 堆 |
| **石子上限** | 每堆最多 5~15 颗 |
| **先手/后手** | 选择玩家或 AI 先走 |

## 数学背景

### 核心定理

**尼姆和 (nim-sum) = 所有堆石子数的 XOR**

```
nim-sum = 0 → 必败局面（先手必输）
nim-sum ≠ 0 → 必胜局面（先手必赢）
```

### Sprague-Grundy 定理（1935）

> **任何一个无偏组合博弈都等价于一个尼姆堆。**

这是组合博弈论的基石定理。它意味着：
- 所有双人轮流走、信息完全、无随机性的博弈
- 都可以化简为 XOR 运算
- 可以用尼姆策略求解

### 必胜策略

1. 计算当前 nim-sum
2. 若 nim-sum = 0，只能随机走（必败，等对手犯错）
3. 若 nim-sum ≠ 0，找一堆满足 `pile[i] ⊕ nim-sum < pile[i]`
4. 从该堆取走 `pile[i] - (pile[i] ⊕ nim-sum)` 颗
5. 使 nim-sum 变为 0
