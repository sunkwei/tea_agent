# 🎮 Tetris — ANSI 终端俄罗斯方块

纯 Python 终端版俄罗斯方块，零外部依赖。

## 快速开始

```bash
python demo/tetris/tetris_ansi.py
```

## 操作

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转 |
| ↓ | 软降（加速下落） |
| 空格 | 硬降（直接落底） |
| p | 暂停/继续 |
| q | 退出 |

## 特性

- 7 种标准方块 (I/O/T/S/Z/J/L)
- 旋转 + Wall Kick（贴墙旋转自动偏移）
- 分数系统 + 逐级加速（每消 10 行升一级）
- 下一个方块预览
- 彩色 ANSI 渲染，终端原生体验
- 跨平台（Windows/Linux/macOS）

## 计分

| 消行数 | 基础分 |
|--------|--------|
| 1 | 100 |
| 2 | 300 |
| 3 | 500 |
| 4 (Tetris!) | 800 |

实际得分 = 基础分 × 当前等级
