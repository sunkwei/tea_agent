# 项目知识库 (Project KB)

存储位置: `$CURR_PATH/.tea_agent_run/`

## 与用户知识库的区别
| | 项目 KB | 用户 KB |
|------|---------|---------|
| 路径 | `$CURR_PATH/.tea_agent_run/` | `$HOME/.tea_agent/kb/` |
| 作用域 | 当前项目 | 跨项目通用 |
| 内容 | 架构、约定、技术决策 | 偏好、经验、通用知识 |
| 工具 | `toolkit_file` | `toolkit_kb` |
| Git | 追踪 | 不追踪 |

> $CURR_PATH = app 启动时的当前目录，即项目根目录
